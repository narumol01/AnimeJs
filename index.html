Anime.js

JavaScript animation engine

คืออะไร ?

Anime.js คือ lightweight JavaScript animation library anime.js เป็นไลบรารีแอนิเมชั่น JavaScript น้ำหนักเบาที่ให้คุณสร้างแอนิเมชั่นที่ซับซ้อนด้วย API อันทรงพลังเพียงตัวเดียว มันทำงานได้กับอะไรก็ได้บนเว็บ: CSS properties, SVG, DOM attributes and JavaScript Objects.

anime.js ทำให้ง่ายต่อการสร้างแอนิเมชั่นที่มีเลเยอร์และซับซ้อน และให้คุณควบคุมจังหวะเวลาและทริกเกอร์เหตุการณ์ได้อย่างเต็มที่ เพื่อให้เคลื่อนไหวได้อย่างแม่นยำว่าคุณต้องการอย่างไรและเมื่อไหร่ รองรับเบราว์เซอร์ล่าสุดบางตัว และมีตัวอย่างและการสาธิตมากมายเพื่อให้คุณเคลื่อนไหวได้ในเวลาไม่นาน เคลื่อนไหวทุกสิ่งที่คุณต้องการอย่างง่ายดายด้วย anime.js

มีไลบรารีแอนิเมชั่น JavaScript มากมาย แต่ Anime.js เป็นหนึ่งในสิ่งที่ดีที่สุด ใช้งานง่าย มี API ขนาดเล็กและเรียบง่าย และมีทุกสิ่งที่คุณต้องการจากเอ็นจิ้นแอนิเมชั่นสมัยใหม่ ไลบรารีมีขนาดไฟล์เล็กและรองรับเบราว์เซอร์สมัยใหม่ทั้งหมด รวมถึง IE/Edge 11+


แหล่งอ้างอิง: animejs.com

ทำอะไรได้ ?

Staggering ระบบการสะท้อนภายในของ anime ทำให้การติดตามและภาพเคลื่อนไหวที่ทับซ้อนกันเป็นเรื่องง่าย สามารถใช้ได้ทั้งกับ timings และ properties. เราจะสามารถทำงานได้อย่างง่ายขึ้นและสามารถลดเวลาในการทำงานให้สามารถลดลงได้ดีด้วยกสรใช้ library anime.js

Layered CSS transforms องค์ประกอบเดียวคือสิ่งที่คุณต้องการ.. Animate multiple CSS แปลงคุณสมบัติด้วยการกำหนดเวลาต่างกันไปพร้อมกันในองค์ประกอบ HTML เดียวได้ โดยanime.js หากไม่มีการระบุระยะเวลาในคีย์เฟรม ระยะเวลาคีย์เฟรมแต่ละรายการจะเท่ากับระยะเวลาทั้งหมดของภาพเคลื่อนไหวหารด้วยจำนวนคีย์เฟรม

Complete controls and callbacks เวลาคือทุกสิ่ง.. สามารถตั้งค่าการ เล่น หยุดชั่วคราว ควบคุม ย้อนกลับ และทริกเกอร์เหตุการณ์พร้อมกันโดยใช้ฟังก์ชันเรียกกลับและการควบคุมในตัวที่สมบูรณ์

Works with CSS, SVG, DOM attributes and JavaScript Objects ไม่ว่าจะเป็น HTML, JS, CSS, SVG. ก็สามารถใช้งานกับ anime ได้ทั้งสิ้น Anime.js ใช้งานได้กับเว็บอะไรก็ได้ แอตทริบิวต์ CSS, SVG, DOM และออบเจ็กต์ JavaScript: ทำให้ทุกอย่างเคลื่อนไหวด้วย API ที่รวมเป็นหนึ่งเดียว

แหล่งอ้างอิง: animejs.com

การใช้งานเป้าหมายที่เป็นวัตถุ JavaScript

คลิกเพื่อลอง

เงื่อนไข
วัตถุ JavaScript ที่มีคุณสมบัติอย่างน้อยหนึ่งรายการที่มีค่าตัวเลข

ประเภท ค่าเริ่มต้น กรณี
วัตถุ ไม่มีค่า targets: myObjectProp
var logEl = document.querySelector('.battery-log');

var battery = {
charged: '0%',
cycles: 120
}

anime({
targets: battery,
charged: '100%',
cycles: 130,
round: 1,
easing: 'linear',
update: function() {
logEl.innerHTML = JSON.stringify(battery);
}
});

'scale' -

translateX: 250,
scale: 2,
rotate: '1turn'
});

การใช้งานค่าที่เป็น ค่าตามฟังก์ชัน

เงื่อนไข
CSS รับค่าที่แตกต่างกันสำหรับทุกเป้าหมายและคุณสมบัติของแอนิเมชั่น
ฟังก์ชันยอมรับ 3 อาร์กิวเมนต์

อาร์กิวเมนต์ ข้อมูล
เป้าหมาย องค์ประกอบเป้าหมายที่เคลื่อนไหวได้ในปัจจุบัน
ตำแหน่ง ดัชนี(ตำแหน่ง)ขององค์ประกอบเป้าหมายแบบเคลื่อนไหว
ความยาวของเป้าหมาย จำนวนเป้าหมายที่เคลื่อนไหวได้ทั้งหมด
anime({
targets : '.function-based-values .el3',
translateX: function (el3) {
return el3.getAttribute('data-x');
},
translateY: function (el3, i) {
return -250 + (-150 * i);
},
scale: function (el3, i, l) {
return (l - i) + .1;
},
rotate: function () { return anime.random(-360, 360); },
borderRadius: function () { return ['50%', anime.random(10, 35) + '%']; },
duration: function () { return anime.random(1200, 1800); },
delay: function () { return anime.random(0, 400); },
direction: 'alternate',
loop: true
});

การใช้งานคีย์เฟรมที่เป็น คุณสมบัติคีย์เฟรม

เงื่อนไข

[หากไม่มีการระบุระยะเวลาในคีย์เฟรม ระยะเวลาคีย์เฟรมแต่ละรายการจะเท่ากับระยะเวลาทั้งหมดของภาพเคลื่อนไหวหารด้วยจำนวนคีย์เฟรม]

เงื่อนไข

'y' ตามแกน y

rotateZ: anime.stagger([0, 90], { grid: [14, 10], from: 'center', from: 'x' }),

});
